Hangman

Hangman is a game played between two players, one person who chooses a word and another who takes turns guessing letters. If the guesser chooses 6 letters that are not in the word before completing the word, they lose the game. If they correctly guess all letters in the word, they win the game.

Football API

An API of made up of 3 different databases (football teams leagues and players) built as a NoSQL databases in MongoDB, utilizing the Mongoose ODM library and Express.js web app framework. Each database follows a REST architectural structure allowing for updates, insertions, and deletions.
